# 税务申报

读取文件夹中所有文件，从文件名中获取相关数据并生成csv文件，用于税务申报。

### 文件名格式

```bash
# 日期-时间-价格x计算比例-类型-描述.jpg
yyyyMMdd-HHmmss-10000x0.5-类型-描述.jpg
# 日期-时间-价格x分期信息1/3-类型-描述.jpg
yyyyMMdd-HHmmss-10000x1_3-类型-描述.jpg
```

### 类型

- 給料賃金
- 外注工賃
- 減価償却費
- 貸倒金
- 地代家賃
- 利子割引料
- 租税公課
- 荷造運賃
- 水道光熱費
- 旅費交通費
- 通信費
- 広告宣伝費
- 接待交際費
- 損害保険料
- 修繕費
- 消耗品費
- 福利厚生費
- 雑費
- 医療費
- 税
- 保险

## 使用

```bash
npm run start
```
